## La Philosophie du Cours : "Intuition-First"
Dans ce cours, nous suivons une philosophie "L'intuition d'abord". Comme le disent si bien Les experts en Intelligence Artificielle: *le domaine de l'IA de langage évolue si vite que la frustration peut vite s'installer si l'on essaie de tout mémoriser*. 

**Ma promesse de professeur** : Nous n'allons pas nous noyer dans les équations mathématiques complexes dès le départ. Nous allons d'abord construire des **images mentales** fortes. Nous allons voir l'IA comme une cathédrale que l'on bâtit brique par brique. Une fois que vous aurez l'intuition du mécanisme, le code deviendra une évidence.


## La Carte de votre Voyage : Les Trois Piliers
Le cursus est structuré autour de trois piliers fondamentaux que nous explorerons pendant 15 semaines :

1.  **Les Fondements (Semaines 1-5)** : Nous partirons de la "poussière" des mots pour comprendre comment on fabrique un cerveau numérique. Nous étudierons les tokens, les embeddings et le cœur nucléaire de l'IA moderne : l'architecture **Transformer**.

2.  **La Science des LLM (Semaines 6-10)** : Nous apprendrons à faire dialoguer l'IA avec le monde réel. C'est ici que nous découvrirons la recherche sémantique, le **RAG** (donner une mémoire à l'IA) et la **multimodalité** (donner des yeux à l'IA). 

3.  **L'Ingénierie des LLM (Semaines 11-14)** : C'est la phase de forge. Vous apprendrez le **Fine-tuning**, l'alignement éthique (**DPO**), les agents et le déploiement en production. C'est là que vous passerez du statut d'étudiant à celui d'expert prêt pour le marché du travail.

## Mes 5 Conseils pour Réussir

> [!TIP]
Pour tirer le meilleur de ces 15 semaines, je vous demande de suivre ces préceptes :

1.  **Ne soyez pas des "Copier-Colleurs"** : Le code est disponible sur GitHub, mais la vraie connaissance naît quand vous modifiez un paramètre et que vous observez la réaction du modèle. 
>> [!IMPORTANT]
**Je dois insister :** cassez le code, faites des erreurs, c'est ainsi que l'on apprend !

2.  **Pratiquez le "GPU-Poor" d'esprit** : Nous avons conçu tous les laboratoires pour qu'ils tournent sur une carte **NVIDIA T4 (16 Go VRAM)** via Google Colab. Vous n'avez pas besoin d'un supercalculateur pour être un génie. L'ingéniosité technique consiste à faire beaucoup avec peu.

3.  **Visualisez tout** : Utilisez les diagrammes du cours comme des cartes. Si vous ne pouvez pas dessiner le flux d'un token à travers une couche d'attention, c'est que vous ne l'avez pas encore compris.

4.  **L'Éthique n'est pas une option** : À chaque étape, demandez-vous : "Quels sont les biais de ce modèle ?". Une IA puissante mais injuste est un échec technologique.

5.  **Aidez la communauté** : L'open-source est le moteur de cette révolution. Si vous trouvez un bug ou une amélioration, partagez-le.
